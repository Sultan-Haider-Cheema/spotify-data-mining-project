# 🎵 Spotify Data Mining Project

## 📌 Overview
This project analyzes Spotify song data using data mining and machine learning techniques.

## 🎯 Objectives
- Predict song genre using audio features
- Analyze factors influencing popularity
- Discover hidden patterns using clustering

## 📊 Dataset
- ~26,000 songs
- Audio features such as danceability, energy, tempo, etc.

## ⚙️ Methods Used
- Decision Tree
- Random Forest
- Linear Regression
- K-means Clustering

## 📈 Results
- Decision Tree Accuracy: 40%
- Random Forest Accuracy: 54%
- Regression R²: 0.073

## 📂 Project Structure
- `data/` → dataset files
- `scripts/` → R scripts
- `output/` → plots and results

## 🚀 How to Run
1. Open project in RStudio
2. Run scripts in order:
   - 01_data_loading.R
   - 02_eda.R
   - 03_preprocessing.R
   - 04_modeling.R
   - 05_clustering.R
